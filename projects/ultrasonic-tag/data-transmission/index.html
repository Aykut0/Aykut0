Overview

This module is responsible for receiving SPI data from the STM32 microcontroller and transmitting the processed data to a Firebase Realtime Database over WiFi using the ESP32. It operates in SPI slave mode and ensures real-time data handling, optimized for low latency and high reliability.

Key Features

  • WiFi connection with retry logic using FreeRTOS events
  • SPI slave configuration to receive 700-sample ADC data packets
  • Firebase HTTPS communication using POST and PUT methods
  • Efficient use of FreeRTOS tasks and semaphores
  • Timestamped data uploads and live data update via a 'latest' node
